# GcoresOriginalGameStar

## requirements
```
pip3 install -r requirements.txt
```

## how to use
```
# update starts of all games
python3 get_star.py --update all
# generate the html plot
python3 gen_html.py
# print specific BOOOM type
python3 get_star.py --print 24SideEffect
```
